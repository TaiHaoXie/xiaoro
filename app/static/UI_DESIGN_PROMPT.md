## 🎯 设计风格

- **整体风格**：简约高级、干净通透，参考Apple官网和Notion的设计语言
- **配色方案**：莫兰迪色系（低饱和度、温柔柔和）
  - 主色：#8B9DC3（灰蓝）或 #9CA8B8（高级灰）
  - 背景：#F7F7F5（暖米白）或 #FAF9F6（米白）
  - 次要色：#D8CFC4（燕麦）、#C5B9A8（卡其）、#B5A89F（灰粉）
  - 强调色：#A8B8C8（雾霾蓝）用于按钮和激活状态
  - 文字：#3D3D3D（深灰，非纯黑）
  - 边框：#E8E6E3（浅灰）

- **字体**：
  - 中文：-apple-system, "PingFang SC", "Microsoft YaHei"
  - 英文/数字：SF Pro、Inter、system-ui
  - 字重：300/400（Regular）、500（Medium）

- **圆角**：12px-16px（大圆角，柔和）
- **阴影**：极浅阴影 `box-shadow: 0 2px 12px rgba(0,0,0,0.04)`
- **间距**：宽松布局，大量留白
